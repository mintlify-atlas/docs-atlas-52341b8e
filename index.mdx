## What is Los Guayacos?

**Los Guayacos** is a full-stack food ordering web application for an authentic Ecuadorian restaurant. It lets restaurant staff and customers:

- Browse dishes categorized by nationality
- Create orders with main items and extras
- Track and dispatch orders through their lifecycle
- Manage customer records by phone number
- Authenticate securely with email/password or OAuth

## Tech stack

| Layer | Technology |
|---|---|
| Framework | Next.js 15 (App Router) |
| Language | TypeScript |
| Database | PostgreSQL via Prisma ORM |
| DB Hosting | Neon (serverless Postgres) |
| Styling | Tailwind CSS v4 |
| Runtime | Edge (Vercel / Cloudflare Pages) |
| Package manager | pnpm |
