---
title: 'Customers'
description: 'Retrieve the list of registered customers in the Los Guayacos system.'
---

## Overview

The customers endpoint returns all registered customers, ordered alphabetically by first name. Each customer record includes personal contact details and an optional nationality association.

## Endpoint

```
GET /api/customers
```

## Query parameters

This endpoint does not accept query parameters. All customers are returned in a single response, sorted by `first_name` ascending.

## Response fields

<ResponseField name="id" type="integer" required>
  Unique identifier for the customer, auto-incremented.
</ResponseField>

<ResponseField name="first_name" type="string">
  Customer's first name. May be `null` if not provided at registration.
</ResponseField>

<ResponseField name="last_name" type="string">
  Customer's last name. May be `null` if not provided at registration.
</ResponseField>

<ResponseField name="phone_number" type="string" required>
  Customer's phone number. Unique across all customers — used as the primary contact identifier.
</ResponseField>

<ResponseField name="nationality_id" type="integer">
  Foreign key referencing a [Nationality](/api-reference/resources/nationalities) record. May be `null` if nationality was not specified.
</ResponseField>

## Example request

<RequestExample>

</RequestExample>
