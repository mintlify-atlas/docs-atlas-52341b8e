title: 'Menu items'
description: 'Retrieve all menu items and their available variants, including pricing and availability status.'
---

## Overview

The menu items endpoint returns the full catalog of items offered at Los Guayacos. Each item includes its category, description, image, active status, and a nested array of `item_variants` — the purchasable size or preparation options with individual prices.

## Endpoint

```
GET /api/menu_items
```

## Query parameters

This endpoint does not accept query parameters. All menu items are returned in a single response, each with their associated variants included.

## Response fields

<ResponseField name="id" type="integer" required>
  Unique identifier for the menu item, auto-incremented.
</ResponseField>

<ResponseField name="category_id" type="integer" required>
  Identifier of the category this item belongs to (e.g., burgers, drinks, sides).
</ResponseField>

<ResponseField name="name" type="string" required>
  Display name of the menu item.
</ResponseField>

<ResponseField name="description" type="string" required>
  Full description of the menu item as shown to customers.
</ResponseField>

<ResponseField name="img_url" type="string">
  URL of the item's image. May be `null` if no image has been uploaded.
</ResponseField>

<ResponseField name="is_active" type="boolean" required>
  Whether the item is currently available for ordering. Defaults to `true`.
</ResponseField>

<ResponseField name="item_variants" type="object[]" required>
  List of purchasable variants for this item (e.g., sizes or preparation styles). Always included in the response.

  <Expandable title="item_variants properties">
    <ResponseField name="id" type="integer" required>
      Unique identifier for the variant, auto-incremented.
    </ResponseField>

    <ResponseField name="item_id" type="integer" required>
      Foreign key referencing the parent `MenuItems` record.
    </ResponseField>

    <ResponseField name="variant_name" type="string" required>
      Label for this variant, such as `"Small"`, `"Large"`, or `"With cheese"`.
    </ResponseField>

    <ResponseField name="price" type="number" required>
      Price of this variant in the store's currency, as a floating-point number.
    </ResponseField>

    <ResponseField name="is_active" type="boolean" required>
      Whether this specific variant is currently available. Defaults to `true`.
    </ResponseField>
  </Expandable>
</ResponseField>

## Example request

<RequestExample>


</RequestExample>
